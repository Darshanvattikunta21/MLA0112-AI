# Python and Prolog Examples

This repository contains small Python and Prolog programs for common AI and logic topics.

## Python Files

- `bfs.py` - Breadth-first search
- `dfs.py` - Depth-first search
- `decisiontree.py` - Simple decision tree classifier
- `astar_search.py` - A* search
- `minimax.py` - Minimax algorithm

## Prolog Files

- `backward_chaining.pl` - Backward chaining inference
- `best_first_search.pl` - Best-first search
- `bird_flight.pl` - Bird flight rules
- `dieting_system.pl` - Diet suggestion system
- `family_tree.pl` - Family tree relations
- `forward_chaining.pl` - Forward chaining inference
- `fruit_color_backtracking.pl` - Fruit color backtracking
- `medical_diagnosis.pl` - Medical diagnosis rules
- `monkey_banana.pl` - Monkey banana problem
- `name_dob.pl` - Name and date of birth facts
- `planets_db.pl` - Planets database
- `student_teacher_subcode.pl` - Student, teacher, and subject code database
- `sumof_int.pl` - Sum of integers
- `towers_of_hanoi.pl` - Towers of Hanoi
